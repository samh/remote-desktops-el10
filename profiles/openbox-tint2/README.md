# Openbox plus tint2 profile

This profile represents a lightweight X11 desktop based on:

- `openbox`
- `obconf`
- `dunst`
- `tint2`

This profile also provides:

- Openbox config for `~/.config/openbox/`
- tint2 config for `~/.config/tint2/`
- a TurboVNC WM template and compatibility `xstartup` file

Current defaults are tuned for a minimal remote-desktop session:

- one desktop
- a bottom `tint2` taskbar
- `dunst` notifications
- a small right-click root menu
